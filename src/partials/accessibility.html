Accessibility statement

The operator of the website https://embroidery-wds4.vercel.app undertakes to ensure the accessibility of its website in accordance with the requirements of the law and international standards, in particular WCAG 2.1 level AA.

Objective

We strive to ensure that all users, including people with disabilities, have full and equal access to the information and functionality of our website.

Compliance status

This website fully complies with the requirements of WCAG 2.1 level AA. The following aspects were taken into account during development and testing:

  • All images have alternative text (alt).
  • Sufficient contrast between text and background is provided.
  • Forms have appropriate labels for input fields.
  • Navigation is possible using the keyboard with visible focus.
  • Semantic markup is used to improve accessibility.

Tools and methods used

During the development and testing of accessibility, we used:

  • Web Content Accessibility Guidelines (WCAG) 2.1.
  • Automated accessibility testing tools.
  • Testing using the keyboard and assistive technologies.

Date of preparation

This statement was made on April 22, 2025, based on an independent assessment of the website's accessibility.
